# Date Interpretation Rules

## Week-to-Date (and similar "to-date" periods)
When the user asks about **week to date (WTD)**, or any similar "to-date" period (e.g., month to date, quarter to date, year to date), the end of the period is **yesterday** (i.e., the current date minus 1 day), not today. Today's data is considered incomplete and should be excluded unless the user explicitly asks to include it.

Examples:
- "Week to date" → from the start of the current week through **yesterday**
- "Month to date" → from the start of the current month through **yesterday**
- "Quarter to date" → from the start of the current quarter through **yesterday**
- "Year to date" → from the start of the current year through **yesterday**
